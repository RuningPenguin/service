# 创建新应用

新应用：nest g app 项目名
新库：nest g lib 库名

# 创建新module

nest g mo module
nest g co controller
nest g s service

nest g res 模块
nest g resource modules/sys/模块

--no-spec 不生成 spec 文件
--no-crud 不生成 CRUD

nest g resource modules/role --no-spec

# 创建自定义模版module

nest g resource abc --collection nestjs-resource-schematics
